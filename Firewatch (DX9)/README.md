Firewatch
=========

Fixed
-----
- Lights
- Shadows
- Specular highlights
- Glow around lights
- Automatically adjust crosshair depth

Installation
------------
1. Extract the zip file to the game directory

2. Right click on the game in Steam -> Properties -> Set Launch Options ->
   enter "-force-d3d9" (without the quotes)
